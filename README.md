# Noita-Save-Editor
This is a save editor for the rogue like Noita. It is allows you to edit your save without disabling achivements, even though this can be accomplished with a mod. This lets you edit perks, wands, wand stats spells on the wand, it's a sandbox mode but you can edit your saves however you'd wish. Some perks don't work, like +50% HP.



Download noita_wand_editor_gui.py and run it, that'll run the program which will let you edit your save.
