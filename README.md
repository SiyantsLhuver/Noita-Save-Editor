Noita Save Editor is a graphical save editor for Noita that lets you modify perks, wands, spells, and other save data without requiring gameplay mods.


## Known Issues

- +50% HP does not currently function correctly.


## Features

- Edit wands
- Edit player stats
- Edit perks
- Edit spells
- Preserve Steam achievements
- Clean UI 


## Installation

1. Install Python 3.11 or newer.
2. Download `noita_wand_editor_gui.py`.
3. Double-click it or run:


## Usage 

1. Press "Open.."
2. Go to "C:\Users\USERNAME\AppData\LocalLow\Nolla_Games_Noita\save00"
3. Enjoy

## Supports:
- Windows
- Latest Steam version of Noita
